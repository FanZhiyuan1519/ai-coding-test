# 招标采购比对系统前端技术文档

## 1. 概述

本文档描述招标采购比对系统前端页面的技术实现方案。

## 2. 技术选型

| 项 | 选型 | 版本 |
|----|------|------|
| 标记语言 | HTML5 | - |
| 样式框架 | Tailwind CSS | CDN (3.x) |
| 图标库 | Material Symbols | CDN |
| 字体 | Public Sans + 系统默认 | CDN |
| 脚本语言 | 原生 JavaScript | ES6+ |
| HTTP 通信 | Fetch API + XMLHttpRequest | - |

## 3. 页面清单

| 页面 | 路径 | 功能 |
|------|------|------|
| 任务列表页 | `/index.html` | 任务 CRUD、状态轮询、筛选分页 |
| 任务详情页 | `/task.html` | 文件上传、触发比对、进度展示 |
| 报告详情页 | `/report.html` | 相似度矩阵、风险项、报价分析 |

## 4. 目录结构

```
frontend/
├── index.html          # 任务列表页（首页）
├── task.html           # 任务详情页
├── report.html         # 报告详情页
├── css/
│   └── custom.css      # 自定义样式（预留）
└── js/
    ├── api.js          # API 调用封装
    ├── tasks.js        # 任务列表逻辑
    ├── task.js         # 任务详情逻辑
    └── report.js       # 报告详情逻辑
```

## 5. API 接口对照

| 前端行为 | HTTP 方法 | 后端接口 | 请求/响应 |
|---------|----------|---------|----------|
| 获取任务列表 | GET | `/api/v1/tasks` | Query: page, limit |
| 获取任务详情 | GET | `/api/v1/tasks/{id}` | - |
| 创建任务 | POST | `/api/v1/tasks` | Body: {name} |
| 删除任务 | DELETE | `/api/v1/tasks/{id}` | 204 |
| 上传文件 | POST | `/api/v1/tasks/{id}/documents` | FormData: supplier_name, file |
| 触发比对 | POST | `/api/v1/tasks/{id}/run` | 202 |
| 获取报告 | GET | `/api/v1/reports/task/{id}` | - |

## 6. 页面交互流程

```
index.html (列表)
    ├── 新建任务 → POST /tasks → 刷新列表
    ├── 查看详情 → task.html?id={id}
    ├── 删除任务 → DELETE /tasks/{id} → 刷新列表
    └── 状态轮询 → GET /tasks/{id} (每10s)

task.html (详情)
    ├── 上传文件 → POST /documents → 更新文件列表
    ├── 触发比对 → POST /run → 开始轮询进度
    ├── 轮询进度 → GET /tasks/{id} (每10s)
    └── 查看报告 → report.html?task_id={id} (status=completed时)

report.html (报告)
    ├── 获取报告 → GET /reports/task/{id}
    └── 返回详情 → task.html?id={task_id}
```

## 7. 轮询机制

- **轮询间隔**：10 秒
- **轮询对象**：`status = "processing"` 的任务
- **停止条件**：`status` 变为 `completed` 或 `failed`

## 8. 状态映射

| 后端状态 | 前端显示 | 样式 |
|---------|---------|------|
| pending | 等待中 | 灰色标签 |
| processing | 进行中 / 提取内容中 | 青色 + 进度条 |
| completed | 比对完成 | 绿色标签 |
| failed | 处理失败 | 红色标签 |

## 9. 风险等级颜色

| 等级 | 颜色 | 说明 |
|------|------|------|
| high | 红色 | 高风险 |
| medium | 黄色 | 中风险 |
| low | 绿色 | 低风险 |

## 10. 相似度矩阵颜色

| 范围 | 颜色 | 说明 |
|------|------|------|
| ≥ 70% | 红色背景 | 高风险 |
| 30-69% | 黄色背景 | 中风险 |
| < 30% | 绿色背景 | 低风险 |

## 11. 错误处理

| 错误类型 | 处理方式 |
|---------|---------|
| HTTP 4xx | 显示后端返回的 `detail` 消息 |
| HTTP 5xx | 显示"服务器内部错误" |
| 网络错误 | 显示"网络连接失败" |

## 12. 约束条件

1. 不实现登录/权限控制
2. 不实现导出 PDF 功能
3. 不实现分享功能
4. 不实现暗色模式
5. 不实现多余页面（首页、项目管理、专家库、系统设置等）
6. 仅支持 .txt/.pdf/.docx 文件上传
